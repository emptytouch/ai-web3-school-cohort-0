# Agent 实验清单

## 实验 1：DAO 提案研究 Agent（只读版）

**目标**：构建一个只读的 DAO 提案研究 Agent，输出投票前检查清单。

**流程**：
1. 读取提案正文
2. 检索论坛讨论
3. 总结支持和反对理由
4. 标记缺失信息
5. 输出投票前检查清单

**约束**：不直接投票。输出中必须明确：
- 用了哪些来源
- 哪些结论证据不足
- 是否发现治理或资金风险
- 如果用户要投票，还需要人工检查什么

**状态**：待开始

**目标**：在只读版基础上，增加授权 + simulation + 交易草稿生成。
