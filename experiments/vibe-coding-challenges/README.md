# Vibe Coding 挑战

## 挑战 1：安装并配置 Vibe Coding 工具

**目标**：选一个工具（Claude Code / Codex CLI / OpenCLI），完成从零到能用的配置。

**完成标准**：
- [ ] 能启动工具并读取当前项目
- [ ] 已配置必要的模型、API key、本地工具权限
- [ ] 完成一个只读任务（解释目录结构 / 搜索文件 / 总结代码）
- [ ] 完成一个低风险小改动（补注释 / 生成脚本 / 修改测试）
- [ ] 保留验证记录：工具版本、配置方式、执行命令、关键输出

**状态**：待开始

---

## 挑战 2：AI Coding 完整工程闭环

**目标**：选一个小功能，让 AI Coding Agent 完成一轮完整工程闭环。

**流程**：
1. 写清楚任务边界和不能改的文件
2. 让 Agent 搜索相关代码并给出计划
3. 让 Agent 做最小 patch
4. 运行测试或构建
5. 查看 git diff，手动审查结果
6. 要求 Agent 写一段 PR summary 和验证说明

**完成后记录**：哪些信息给少了，哪些测试缺失，哪些改动需要人类判断。

**状态**：待开始
