## 实验：文档问答 + 工具调用最小框架对比

**目标**：同一任务用两种方式实现，看清框架到底帮了什么。

**任务**：基于检索结果的文档问答 + JSON 结构化输出

**实现 A：直接调用模型 API**
- 用户问题 + 检索结果 + JSON schema 输出
- 自己处理重试、错误处理、trace

**实现 B：使用框架加入**
- tool schema
- trace
- 失败重试
- 测试样本

**对比维度**：
- 哪个版本更容易读懂？
- 哪个版本更容易加工具？
- 哪个版本更容易定位错误？
- 哪个版本更容易写回归测试？
