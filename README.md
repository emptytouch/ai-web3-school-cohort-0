## 简介

这是 [AI × Web3 School](https://aiweb3.school/) 的个人学习仓库，用于记录学习笔记、每日打卡、实验项目和 Handbook feedback。

## 重要提醒

- 本仓库是 **public** 的，任何人都可以查看
- **不要** 在此仓库中存放任何敏感信息：API key、私钥、助记词、密码、未公开联系方式等
- 学习过程中的 proof-of-work、笔记、实验代码都可以放这里
