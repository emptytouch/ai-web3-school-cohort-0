# 个人学习计划

> 基于 [AI × Web3 School Handbook](https://aiweb3.school/zh/handbook/) 大纲
> 最后更新：2026-05-19

## 学员背景

Web2 前端开发，正在写 DApp demo（含合约）。AI 工具使用有经验但未系统配置 Agent/MCP。

## 学习路径

根据 Handbook 的四层地图，结合你的背景，建议按以下优先级学习：

### 阶段一：AI 基础补强（第 1-2 周）

重点：从"用 AI 写代码"升级到"配置 AI Agent 工作流"

| 周次 | Handbook 章节 | 重点内容 | 每日时间 |
|------|--------------|---------|---------|
| W1 | 大语言模型（LLM）| 理解模型能力边界、上下文窗口 | 30 min |
| W1 | 提示词（Prompt）| 系统化 Prompt 设计，不只是聊天 | 30 min |
| W1 | 上下文（Context）| 上下文工程，长对话管理 | 30 min |
| W2 | 智能体（Agent）| Agent 概念、工具调用、ReAct 模式 | 45 min |
| W2 | 框架（Frameworks）| LangChain/CrewAI/AutoGen 等框架对比 | 45 min |
| W2 | 模型上下文协议（MCP）| MCP 配置、Server/Client 架构 | 60 min |

### 阶段二：Web3 基础深化（第 3-4 周）

重点：结合你的 DApp 实操经验，补全 Web3 知识体系

| 周次 | Handbook 章节 | 重点内容 | 每日时间 |
|------|--------------|---------|---------|
| W3 | 智能合约（Smart Contract）| Solidity 进阶、安全模式 | 45 min |
| W3 | 开发栈（Dev Stack）| Foundry/Hardhat、测试、部署 | 45 min |
| W3 | 账户抽象（Account Abstraction）| ERC-4337、智能合约钱包 | 45 min |
| W4 | 去中心化金融（DeFi）| DeFi 协议、可组合性 | 30 min |
| W4 | 预言机（Oracle）| Chainlink、数据喂价 | 30 min |
| W4 | 安全（Security）| 合约安全、审计基础 | 45 min |

### 阶段三：AI × Web3 Bridge（第 5-7 周）

重点：这是你的核心差异化方向

| 周次 | Handbook 章节 | 重点内容 | 每日时间 |
|------|--------------|---------|---------|
| W5 | 链感知上下文（Chain-aware Context）| 让 Agent 理解链上状态 | 60 min |
| W5 | Web3 工具调用（Web3 Tool Use）| Agent 调用合约、发送交易 | 60 min |
| W5 | 智能体工作流（Agent Workflow）| 多步骤链上操作编排 | 60 min |
| W6 | 智能体钱包（Agent Wallet）| Agent 自主管理资产 | 60 min |
| W6 | 机器支付（Machine Payment）| Agent 间支付、微支付 | 60 min |
| W6 | 结算与托管（Settlement & Escrow）| 链上托管、条件支付 | 60 min |
| W7 | 智能体身份（Agent Identity）| DID、Agent 身份验证 | 45 min |
| W7 | 智能体信任与声誉（Agent Trust & Reputation）| 链上声誉系统 | 45 min |
| W7 | AI 预言机（AI Oracle）| AI 作为链上数据源 | 45 min |

### 阶段四：前沿探索 + Hackathon（第 8-10 周）

| 周次 | Handbook 章节 | 重点内容 | 每日时间 |
|------|--------------|---------|---------|
| W8 | 可验证 AI（Verifiable AI）| ZKML、AI 推理验证 | 60 min |
| W8 | AI 安全（AI Security）| Prompt 注入、Agent 安全 | 60 min |
| W9 | 去中心化 AI（Decentralized AI）| 分布式推理、联邦学习 | 60 min |
| W9 | 治理 AI（Governance AI）| AI 辅助 DAO 治理 | 60 min |
| W10 | 智能体商业（Agentic Commerce）| Agent 经济模型 | 60 min |
| W10 | 开放赛道（Open Track）| Hackathon 项目开发 | 120 min |

## 每日学习节奏（建议）

- **早上（30 min）**：阅读 Handbook 章节 + 笔记
- **晚上（30-60 min）**：实验/编码 + 打卡

- [ ] 第 4 周：DApp demo 完成合约部署
- [ ] 第 7 周：实现 Agent 调用链上合约的 PoC
